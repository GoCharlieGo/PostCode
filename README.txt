Необходимо реализовать сайт для публикации кусков кода и простого текста, возможность комментировать.

Основа:
ASP.NET MVC + SQLServer + Entity Framework

Странички:
CoffeeScrit + jQuery
Bootstrap (Или другой CSS фреймворк)

Полнотекстовый поиск:
Механизмы БД или Lucene

Дополнительные требования:
2 визуальных стиля оформления;
2 языка: английский, русский.
Языки и темы должны переключаться с любой страницы и работать для всего сайта, а также сохраняться для каждого пользователя. 

Регистрация/авторизация. Подтверждение о регистрации по email. 
Незарегистрированные пользователи могут только читать. 

Две роли пользователей:
Администраторы (Редактирование всего контента + удаление/блокировка/сброс пароля пользователя)
Пользователи

Публикации: 
Необходимо использовать язык разметки Markdown
Должны содержать фрагменты кода и текст
Необходима реализация подсветки синтаксиса 
Поддержка языков Java, C#, Ruby, Python

Комментарии одноуровневые, без веток.
Возможность редактировать или удалять свой комментарий. 
Админы могут редактировать и удалять все комментарии

Рейтинг пользователей состоит из:
Рейтинг постов (1-5 звёздочек)
Рейтинг комментариев (+/-) (для отрицательного рейтинга использовать серый цвет, читаемый в обеих темах оформления)

Каждый пост должен содержать название, может содержать теги. Для тегов - реализовать автозаполнение (из списка уже существующих)

На главной странице отображать топ-5 пользователей (со ссылками на профили)
Топ-5 популярных постов по всем языкам и
топ-5 популярных постов по каждому из поддерживаемых языков.

Страница профиля ползователя должна содержать информацю о пользователе (редактируется только пользователем) и список на посты пользователя

Полнотекстовый поиск должен быть доступен с любой страницы сайта и должен искать по записям и комментариям. 
При поиске по комментариям переходить надо на страницу поста, содержащего комментарий.

Дополнения:
Проект должен выглядеть целостно и аккуратно.
Желательно использовать по максимуму готовые решения и библиотеки.
